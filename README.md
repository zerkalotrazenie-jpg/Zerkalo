# 🪞 ЗЕРКАЛО (Аль-Ми'ра)

## Полная система «Зеркало» — Разум, Свет, Отражение

### Что это?
«Зеркало» — это философская система, которая помогает человеку увидеть себя настоящего и найти свой путь.

### Функции:
- 🔍 Поиск работы и мастеров
- 💰 Заработок через задания
- 🛡️ Защита и помощь
- 👑 Управление для Хранителя
- 📱 WebApp интерфейс

### Установка:
1. Скопируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте переменные окружения (BOT_TOKEN)
4. Запустите: `python app.py`
