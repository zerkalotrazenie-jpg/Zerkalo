# 🪞 ЗЕРКАЛО

Полная система автоматизации, помощи и заработка.

## Запуск
1. Установить зависимости: `pip install -r requirements.txt`
2. Настроить переменные окружения: `BOT_TOKEN`
3. Запустить: `python app.py`

## Структура
- `app.py` — Telegram-бот
- `webapp/index.html` — Голосовой интерфейс
- `modules/` — Все бизнес-модули
- `suras/suras.txt` — Суры
